# 🚀 비즈니스 통찰력을 담은 개발자, 임형준입니다.

비즈니스 운영 현장에서 얻은 문제 해결 능력을 바탕으로, 사용자에게 실질적인 가치를 전달하는 웹 서비스를 만듭니다.

## 🛠️ Tech Stacks
- **Frontend:** React, JavaScript (ES6+), SASS/SCSS, Tailwind CSS
- **Backend:** Supabase
- **Design/Video:** Figma, Photoshop, Premiere Pro, After Effects

4. **Global Travel Log** - Tailwind CSS 반응형 웹 프로젝트
5. **Snow-Valley** - Vanilla JS 인터랙션 웹사이트
